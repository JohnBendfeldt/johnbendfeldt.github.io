John Bendfeldt


I love working with people and around technology. I enjoy having technology integrated into my life, whether it’s my smartwatch or Android Auto in my car. From my love of technology, I have learned what makes a great product or a strong brand, whether it’s Google redefining how we do business or Amazon assisting with daily life. It’s important to me, therefore, to see what the newest technologies are and to become familiar with them. I want to be able to assist in creating or sustaining a strong business environment that pushes the limits of what we currently have.
Through years of working in management at Domino’s, I saw how apps and technology have the ability to improve lives, strengthen connections and build stronger businesses. I moved to Austin in 2015 to pursue my passion for web development and integrated technologies. I currently work as a Software Engineer at Volusion helping to maintain platform stability and updating features.

Portfolio


Star Wars
RPG
A Star Wars JavaScript based game that allows users to pick a hero to defeat the other characters. Each character has health points and an attack mode that depends on whether or not they are the hero. Each time there is a clash each character takes damage. Once the health points run out the character is defeated and the user selects another opponent.

Technologies Utilized:
JavaScript, jQuery, CSS animation, HTML5

Jimmy John's
Menu Quiz
A Jimmy John’s Quiz to help trainees learn the menu. Users have 15 seconds to answer each question. If they guess wrong or run out of time the correct answer appears and then moves on to the next question. The number of correct and incorrect answers is displayed at the end announcing whether or not the user is a Menu Master.

Technologies Utilized:
JavaScript, jQuery, CSS, HTML5

John’s Burger Bar
A burger ordering app that keeps track of orders that are placed and eaten using Sequelize with MySQL. The user inputs the name of the burger they ate or want to order and it is displayed in the correct location. Users can delete burgers they no longer want or order them again after they are pushed into the eaten section.

Technologies Utilized:
JavaScript, CSS, HTML5, Express.js, Node.js, Heroku, JawsDB, MySQL, Sequelize

STUMP App
A student meetup app to help groups schedule meeting times and locations. Users can select their availability using Moment.js and find the nearest coffee shop with Wi-Fi using geolocation and Google Places API.

Technologies Utilized:
JavaScript, CSS, HTML5, Geolocation, Firebase, Google API, Moment.js

Train Scheduler
A train scheduling app that utilizes Firebase and Moment.js to schedule train rides based on previous user input. Each train stays in the database and shows when the next one will arrive.

Technologies Utilized:
JavaScript, CSS, HTML5, jQuery, FIrebase, Moment.js

Hangman Game
A Hangman game that uses jQuery to display the letters that users guess. Each time the user wins the word is revealed and victory audio is played. When the user runs out of guesses the correct word appears and a sad song plays.

Technologies Utilized:
JavaScript, CSS, HTML5, jQuery
