Star Wars
RPG
A Star Wars JavaScript based game that allows users to pick a hero to defeat the other characters. Each character has health points and an attack that depends on whether or not they are the hero. Each time there is a clash each character takes damage. Once the health points run out the character is defeated and the user selects another opponent.

Technologies Utilized:
JavaScript, jQuery, CSS animation, HTML5

Jimmy John's
Menu Quiz
A Jimmy John’s Quiz to help learn the menu. Users have 15 seconds to answer each question. If they guess wrong or run out of time the correct answer appears and then moves on to the next question. The amount of correct and incorrect answers display at the end announcing whether or not the user is a Menu Master.

Technologies Utilized:
JavaScript, jQuery, CSS, HTML5
